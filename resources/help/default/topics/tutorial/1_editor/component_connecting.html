Connecting components with cables

Again, we start with choosing the right tool. Choose Cable tool in menu using left mouse button.

Connecting without interface selection

Click the firts component with left mouse button. Then click the second component with the same button. You can see cable making in progress on the following image. First free component interface is chosen using left mouse button.

Connecting with interface selection

If you want to choose interface manually, then use right mouse button instead of the left one as shown on the next image

If you want to cancel cable creation progress, then press any mouse button anywhere else than on the components.
