Event playing

Playing captured events

Start playing with the Play button in Control panel:

Moving in the event list while playing

You can move in the event list as you want during playing. Use the previous, next, first and last buttons in Play controls panel, or use mouse click on table rows.

Speed setting

Playing speed can be set with a slider in Play controls panel. New speed will be applied on next played event. Current animation wil finish in the original speed.

Sequential playing

You can play events sequentially (one by one). This mode will give you time to observe events. Duration of the animation depends on speed and cable delay.

Playing by timestamps

Events can be allso played in recieved times (timestamps). In this mode the animation duration depends on speed and cable delay, but there could be more animations at a time because they were recieved in short time intervals.

Realtime mode

In Realtime mode packets are both recieved and played. Events are played immediately when they are recieved. In this mode the duration of animation depends on speed and cable delay. On the next image the realtime button is marked:
