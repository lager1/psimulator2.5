Data in event list

Each row represets one captured event. Column description follows:

  • Time[s] - timestamp of recieved packet. Always starts from 0,00.
  • From - name of source device.
  • To - name of destination device.
  • Type - packet type.
  • Info - color specific for packet type.

Event detail
